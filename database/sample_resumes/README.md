## How to Use

1. **Upload via App UI** — Log in to the app at http://localhost:5173,
   navigate to Resume Upload, and upload any `.txt` file from this folder.

2. **Test Job Matching** — After uploading, check the AI job recommendations
   to see how the system matches resumes to job descriptions.

3. **Test Recruiter Screening** — As a recruiter, post a job and run bulk
   screening with multiple resumes from this folder to test ranking.

4. **ATS Score Testing** — Each resume is crafted with keywords and structured
   sections to test the ATS scoring engine.
